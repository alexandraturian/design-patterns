jQuery $() function is an example of Builder pattern

jQuery $() is used to create new DOM elements. The API exposes methods with which we can set different parts of an element as we need them, without having to build in advance, at once, the whole HTML string needed for the final element.
  • $() returns a partially constructed element that is not yet part of the DOM tree
  • $() provides a way to manipulate the internal state of an object
  • $() provides a way to get the final result of the construction: appendTo(), insertAfter()
  • Internally, jQuery uses createDocumentFragment
